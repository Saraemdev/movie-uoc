Sobre esta web

Esta PWA forma parte de una práctica de la asignatura Desarrollo front-end avanzado del máster Desarrollo de sitios y aplicaciones web de la UOC Para el desarrollo de este trabajo se han utilizado herramientas y librerías de terceros, así como imágenes prediseñadas, tal y como se detallan a continuación:

Herramientas de desarrollo

  • Desarrollo de la aplicación con Angular
  • Maquetación con Angular Material
  • Carrusel Swiper
  • Consumo de datos mediante la API de The Movie DB
